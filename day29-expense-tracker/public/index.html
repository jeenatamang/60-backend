Expense Tracker API

A RESTful API built with Node.js + Express. All routes below.

Expenses

GET/expensesGet all
GET/expenses?category=foodFilter
GET/expenses?sort=descSort
GET/expenses/:idGet one
POST/expensesCreate
PATCH/expenses/:idUpdate
DELETE/expenses/:idDelete

Extra
